[뉴시스] (사진) 인천공항공사 배드민턴 코리아리그 후원 협약식

작성일
2017.10.17
조회수
234

인천국제공항 제2여객터미널 자원봉사자 지원기회 확대를 위해 아래와 같이 접수기간을 연장하오니, 자원봉사를 희망하시는 여러분들의 적극적인 참여를 부탁드립니다.

  • 선발전형 : 여객안내분야, 시설안내분야
  • 선발절차 : 지원서접수 => 서류전형 => 면접전형 => 최종선발
  • 접수기간 : 2017.9.6(수) ~ 10.13(금) (기존 9.28 마감에서 접수기간 연장)
  • 접수방법 : 온라인 ( http://iiac.saramin.co.kr ) 또는 우편 접수

* 자세한 내용은 붙임의 선발공고문을 참고하시기 바랍니다.
